# HotelMngmntproj

A simple Java project with a GUI and MySQL connectivty. The Database is structured around Room No. using SELECT and UPDATE queries.
Meant to be used by Hotel Staff.
Program can:
* Book Room
* Make a reservation
* Check Out
* View Occupied and Unoccupied rooms
